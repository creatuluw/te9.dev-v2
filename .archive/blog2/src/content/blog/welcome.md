## Welcome!

This is my first blog post using the Svelta markdown publishing system.

### Features

- Markdown support with frontmatter
- Automatic reading time calculation
- Category and tag organization
- RSS feed support
- Dark mode support

### Getting Started

Simply create markdown files in `src/content/blog/` with the required frontmatter fields:

- `title`: The post title
- `date`: Publication date (YYYY-MM-DD)
- `category`: The post category
- `author`: Author identifier
- `cover`: Path to cover image
- `excerpt`: Short description

That's it! Your blog post is ready to be published.
