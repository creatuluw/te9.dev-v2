INSTALLATION & SETUP

How do I install LLMNGN? Run: npx degit hoipippeloi/llmngn.xyz/.opencode .opencode && cd .opencode && bun install && cd ..
Do I need to configure anything? No. Works out of the box. Customize in .opencode/plugins/llmngn.json if needed.
How do I verify installation? Start OpenCode, plugin auto-loads. Run llmngn stats.

USAGE

How do I store a decision? llmngn add "Your decision" --type decision
How do I search? llmngn query "your search" — semantic matching finds related concepts
What are context types? Categories: decision, architecture, debt, file_change, task, command — each with retention period

TECHNICAL

Where is data stored? Locally in .lancedb/ folder. No cloud upload.
Is data secure? Yes. Local only. Auto-redacts API keys, passwords, tokens, secrets.
How long is context kept? Architecture: 365d, Decisions: 180d, File changes: 90d, Tasks: 60d, Commands: 30d
Can I use offline? Yes. Everything runs locally. No internet after installation.

TEAM COLLABORATION

Can I share with team? Export and share JSON: llmngn export --output team-context.json
How do teammates import? llmngn import team-context.json
