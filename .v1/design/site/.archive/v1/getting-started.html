Getting Started

Get up and running with LLMNGN in 5 minutes

On this page
  • Install the Plugin
  • Verify Installation
  • Store Your First Decision
  • Use Interactive Commands
  • Next Steps

Step 1: Install the Plugin

Open your terminal or command prompt and navigate to your project folder. Run this single command:

npx degit hoipippeloi/llmngn.xyz/.opencode .opencode && cd .opencode && bun install && cd ..

PowerShell users: Use semicolons instead of &&

npx degit hoipippeloi/llmngn.xyz/.opencode .opencode; cd .opencode; bun install; cd ..

This command installs:

  • Plugin hooks (.opencode/plugins/llmngn.ts)
  • Configuration (.opencode/plugins/llmngn.json)
  • Interactive command (.opencode/commands/llmngn.md)

Step 2: Verify Installation

After installation, verify everything is working:

  1. Start OpenCode in your project
  2. The plugin auto-loads automatically
  3. No additional setup required!

Check the database status:

llmngn stats

Step 3: Store Your First Decision

Now you're ready to start storing context. Open your terminal and run:

llmngn add "This is my first decision note" --type decision

Verify it was stored:

llmngn list
💡 Pro Tip

Always use the --type flag to categorize your context. This helps with organization and search later.

Step 4: Use the Interactive Command

In OpenCode, you can use natural language commands:

/llmngn store that we decided to use Redis for caching
/llmngn show me my latest records
/llmngn find anything about authentication

These commands work within your OpenCode sessions without switching to terminal.
