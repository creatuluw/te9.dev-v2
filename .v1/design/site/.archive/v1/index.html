How it works

Three steps to persistent AI context

1

Store decisions as you code

When you make an architectural choice or complete a task, store it with a simple command.

llmngn add "Using Redis for session cache" --type decision
2

Context auto-injects next session

When you start a new OpenCode session, relevant stored context is automatically loaded. The AI knows your project's history.

3

Search anytime with natural language

Query your context semantically. Find related decisions even if you don't remember exact words.

llmngn query "authentication" --type decision
