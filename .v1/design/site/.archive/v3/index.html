Your AI's
Long-Term Memory

LLMNGN remembers your coding decisions, architectural choices, and patterns across every OpenCode session. No more repeating yourself.

Get Started Try the CLI
Features

Never Lose Context Again

A persistent memory layer for OpenCode that stores, searches, and retrieves your coding decisions instantly.

🧠

Semantic Memory

Stores decisions with vector embeddings. Search by meaning, not just keywords. "Database" finds PostgreSQL, SQL, and data storage.

⚡

Instant Retrieval

Context is automatically injected at session start. Your AI knows your decisions before you explain them.

🏷️

Typed Context

Categorize with types: decisions, architecture, file changes, tasks, debt, commands. Each with appropriate retention periods.

🔒

Local-First

All data stored locally in your project. No cloud upload. Auto-redacts API keys, passwords, and secrets.

🧹

Auto-Cleanup

Expired records auto-delete. Decisions last 180 days, tasks 60 days, commands 30 days. Or clean manually anytime.

💾

Backup & Restore

Export to JSON for backup. Import to restore or share with your team. Your context, your control.

Get Started

Install in 5 Minutes

One command to install. Zero configuration needed.

1

Install the Plugin

Navigate to your project folder and run:

npx degit hoipippeloi/llmngn.xyz/.opencode .opencode && cd .opencode && bun install && cd ..
2

Verify Installation

Start OpenCode in your project. The plugin auto-loads automatically.

llmngn stats
3

Store Your First Decision

Add your first context note:

llmngn add "This is my first decision" --type decision
4

Use in OpenCode

Access LLMNGN with natural language commands:

/llmngn store that we decided to use Redis
Context Types

Organize Your Memory

Six context types, each with appropriate retention periods for healthy context management.

decision 180 days

Important choices that shape your codebase. Architectural decisions, library choices, design patterns.

6 months
architecture 365 days

System design and structural decisions. High-value, long-term context.

1 year
file_change 90 days

Code changes worth remembering. What you modified and why.

3 months
debt 90 days

Technical debt items. Things to revisit later.

3 months
task 60 days

Todos and tasks. Track what needs to be done.

2 months
command 30 days

Important commands run. Reusable command patterns.

1 month
FAQ

Common Questions

Do I need to configure anything?

No! LLMNGN works out of the box with sensible defaults. Customize in .opencode/plugins/llmngn.json if needed.

Where is my data stored?

All data is stored locally in .lancedb/ folder in your project directory. No cloud upload.

Is my data secure?

Yes! Data stays on your machine. The plugin auto-redacts API keys, passwords, and secrets.

Can I use LLMNGN offline?

Yes! Everything runs locally. No internet required after installation.

How does semantic search work?

LLMNGN uses vector embeddings to find conceptually related context. "Database" finds PostgreSQL, SQL, data storage, etc.

Is LLMNGN free?

Yes! Completely free and open source under MIT license.
