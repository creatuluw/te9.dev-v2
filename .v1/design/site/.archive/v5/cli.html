init

llmngn init

Description

Initialize the LLMNGN plugin in your project. Creates the database structure and stores initial usage instructions for future reference.

Usage

llmngn init
✓ Database initialized at .lancedb/ ✓ Usage instructions stored

What It Does

  • Creates .lancedb directory if not exists
  • Initializes vector database tables
  • Stores usage instruction record for future sessions
