How do I install LLMNGN?
Run this in your project folder: npx degit hoipippeloi/llmngn.xyz/.opencode .opencode && cd .opencode && bun install && cd .. This installs the plugin hooks, configuration, and interactive command.
install
Do I need to configure anything?
No! LLMNGN works out of the box with sensible defaults. You can customize settings in .opencode/plugins/llmngn.json if needed.
install
How do I verify it's working?
1. Start OpenCode in your project
2. The plugin auto-loads
3. Run llmngn stats to see database info

How do I store a decision?
Terminal: llmngn add "Your decision" --type decision In OpenCode: /llmngn store that we decided to use Redis
usage
How do I search my context?
Terminal: llmngn query "your search" In OpenCode: /llmngn find anything about authentication
usage
What are context types?
Context types categorize stored information:

• decision — Important choices (180 days)
• architecture — System design (365 days)
• debt — Technical debt (90 days)
• file_change — Code changes (90 days)
• task — Todos (60 days)
• command — Commands run (30 days)
data
Where is my data stored?
All data is stored locally in the .lancedb/ folder in your project directory. Nothing is uploaded to the cloud.
data
Is my data secure?
Yes! Data stays on your machine. The plugin also auto-redacts API keys, passwords, tokens, and secrets before storage.
data
How long is context kept?
Depends on type:
• Architecture: 365 days
• Decisions: 180 days
• File changes: 90 days
• Tasks: 60 days
• Commands: 30 days

Expired records auto-delete with llmngn clean
data
How do I backup my data?
llmngn export --output backup.json Restore with: llmngn import backup.json
team
Can I share context with my team?
Yes! Export and share JSON files: llmngn export --output team-context.json Team members import: llmngn import team-context.json
data
How do I delete records?
Single record: llmngn delete <id> --force All expired: llmngn clean Everything: llmngn purge --force
install
Can I use LLMNGN offline?
Yes! Everything runs locally. No internet required after installation.

Is LLMNGN free?
Yes! LLMNGN is completely free and open source (MIT license). All features are available at no cost.

How does semantic search work?
LLMNGN uses vector embeddings to find conceptually related context, not just keyword matches. Search "database" and find PostgreSQL, Redis, migrations, etc.
