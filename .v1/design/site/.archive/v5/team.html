Stratum: Collective Deposit

TEAM PROTOCOL

Shared context deposits enable institutional memory. Onboard developers through inherited knowledge, not lengthy handovers.

01

Capture Decisions

During planning sessions, store architectural choices as they're made.

llmngn add "React + Node.js + PostgreSQL stack" --type architecture
→
02

Export Team Context

Create portable context backup for sharing.

llmngn export --output team-context.json
03

Share Deposit

Commit to git, share via drive, or transfer directly.

git add team-context.json && git commit -m "Team context"
→
04

Import Inherit

Team members import collective knowledge instantly.

llmngn import team-context.json

The Onboarding Ritual

I

Install LLMNGN

New team member runs the installation command in their project.

npx degit hoipippeloi/llmngn.xyz/.opencode .opencode && cd .opencode && bun install && cd ..
II

Import Team Context

Load the shared knowledge base from team-context.json.

llmngn import team-context.json
III

Search to Learn

Query the inherited memory to understand past decisions.

llmngn query "architecture"
llmngn query "authentication"
IV

Contribute Back

Store new decisions, export periodically, maintain the collective.

llmngn add "Implemented rate limiting" --type file_change
