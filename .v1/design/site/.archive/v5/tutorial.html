Your First
Pathway

Learn to store, search, and persist context across sessions. Your AI will remember. Finally.

Install LLMNGN

Open terminal in your project and run the installation command.

npx degit hoipippeloi/llmngn.xyz/.opencode .opencode && cd .opencode && bun install && cd ..

Verify Installation

Check that LLMNGN is working by viewing database stats.

llmngn stats

Store Your First Decision

Archive an architectural choice. This will survive session death.

llmngn add "Using JWT for stateless authentication" --type decision
Checkpoint You've stored your first memory. It persists.

Add Architecture Context

Store system design decisions with longer retention.

llmngn add "PostgreSQL for primary database with Redis cache" --type architecture

Track a Task

Store todos and pending work items.

llmngn add "Need to implement rate limiting" --type task

View All Records

See everything you've stored so far.

llmngn list
Checkpoint Your context is building. The AI inherits this knowledge.

Search Semantically

Find related concepts, not just keywords.

llmngn query "authentication"

Backup Your Data

Create a portable backup before making changes.

llmngn export --output backup.json

Start a New Session

Close OpenCode and start fresh. LLMNGN auto-injects your context. The AI now knows about your JWT decision without you explaining it again.

Ritual Complete

You've established persistent memory. Your decisions survive.
